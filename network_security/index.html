Network Security Architecture — DMZ + Defense in Depth

Tap each component to understand its role. Notice the two-firewall DMZ design — even a compromised web server can't reach the internal database directly.

Key insight: The DMZ doesn't prevent web server compromise — it LIMITS the blast radius. The internal firewall is what actually protects the database. That's defense in depth: multiple independent security boundaries.
Wireless Security Standards & Attacks

Compare WEP through WPA3, with security/privacy/accountability ratings. Bottom section covers wireless attack types.

Exam tip: WPA2-Enterprise (802.1X + RADIUS) is the correct answer for any organization wireless scenario. WPA3-SAE fixes offline dictionary attacks. WEP is always wrong. Management Frame Protection in WPA3 fixes deauthentication attacks.
