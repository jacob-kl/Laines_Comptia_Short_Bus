Identity, Access & Authentication

// click any card to reveal · covers MFA, federation, directory services, PAM, lifecycle

Core exam pattern: IAM questions test whether you know WHICH protocol or control fits the scenario. SAML = enterprise SSO (XML assertions). OIDC = web app "sign in with X" (JWT). Kerberos = Windows/AD (tickets). FIDO2 = phishing-resistant (origin-bound). PAM = privileged account control (vaulting, JIT, recording).
